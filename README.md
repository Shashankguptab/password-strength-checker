# Password Strength Checker

A simple Python-based Password Strength Checker that evaluates the strength of a password using security validation rules.

## Features

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Classifies password as:
  - Weak
  - Medium
  - Strong

## Technologies Used

- Python

## How It Works

The program analyzes the entered password based on multiple conditions:
- Minimum length
- Presence of uppercase and lowercase letters
- Numeric values
- Special characters

Based on these checks, the system determines the password strength.

## Example

Enter your password:
Password@123

Output:
Strong Password
